# あなたにおすすめの旅は？

旅行先や旅行スタイルを診断できるWebツールです。

## 機能

- 7つの質問に答えて、あなたにぴったりの旅行先を診断
- 現在「世界編」を実装（10の世界的な都市から診断）
- 診断結果には以下が含まれます：
  - 旅行先の美しい画像
  - その旅行先があなたに適している理由
  - 詳細情報へのリンク3つ

## 使い方

1. `index.html`をブラウザで開く
2. 診断カテゴリー（現在は「世界編」）を選択
3. 7つの質問に答える
4. あなたにおすすめの旅行先が表示されます

## 新しい診断カテゴリーの追加方法

`data.js`ファイルの`quizData`オブジェクトに新しいカテゴリーを追加できます。

### 追加例：

```javascript
const quizData = {
    "世界編": {
        // 既存のデータ
    },
    "アジア編": {
        questions: [
            {
                question: "質問文",
                options: [
                    { 
                        text: "選択肢1", 
                        scores: { destination1: 2, destination2: 1 } 
                    },
                    // 他の選択肢...
                ]
            },
            // 他の質問（合計7つ）...
        ],
        destinations: {
            destination1: {
                name: "都市名",
                country: "国名",
                image: "画像URL",
                description: "説明文",
                links: [
                    { text: "リンク名", url: "URL" },
                    // 3つのリンク
                ]
            },
            // 他の目的地...
        }
    }
};
```

### 追加手順：

1. `data.js`を開く
2. `quizData`オブジェクトに新しいカテゴリーを追加
3. `questions`配列に7つの質問を定義
4. 各質問の選択肢に、目的地ごとのスコアを設定
5. `destinations`オブジェクトに目的地の詳細情報を追加
6. ファイルを保存して、ブラウザで確認

## 技術スタック

## カスタマイズ

- 色やデザインは`style.css`で変更可能
- 質問数を変更する場合は、`data.js`の`questions`配列を編集
- 画像は下記のフリー素材から取得していますが、任意の画像URLに変更可能
    1. [Unsplash](https://unsplash.com/)
    2. [photo-ac](https://www.photo-ac.com/)
    3. [阿蘇市ホームページ](https://photo.city.aso.kumamoto.jp)
